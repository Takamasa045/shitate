# Lexicon / Clothing / SK8 芝犬クルー

SK8 芝犬クルー（ふわ柴・渋柴・赤柴・もも柴・黒柴）の共通衣装断片。
スニーカー色・毛色・表情は各キャラの base に残し、**キャップ・パーカー・ショーツ・スケボー**だけを共有する。

---

## #uniform

**用途**: SK8 芝犬クルーの共通ストリートウェア。シリーズショットで衣装を揃えるとき。

**プロンプト断片**:
```
wearing a black baseball cap with red "SK8" box logo on the front,
dark teal hoodie with a small red graffiti-style logo on the chest,
black shorts, white ankle socks,
holding a skateboard in one paw, skateboard has black deck with red lightning-bolt graphics
```

**相性の良い Lexicon**:
- （スタイルはキャラ base 側。flat sticker / cel-shaded は base に固定）
