# 和紙狐 — 三面図

## 用途
正面・側面・背面を並べ、キャラクターの形と配色を確認する。

## 依存ベースバージョン
v1

## 本文プロンプト
```
three-view character sheet showing front, side, and back,
neutral standing pose, full body visible in every view,
consistent proportions, colors, scroll case, bookmark cord, and single tail,
plain light background, generous spacing, no labels
```

## ネガティブプロンプト
```
inconsistent design between views, dynamic action pose, perspective distortion,
cropped body, multiple characters, text labels, captions, decorative frame
```
